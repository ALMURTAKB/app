## نبذة عن المشروع

موقع تسويقي لمنصة **سوبر جات**، منتج من شركة **المرتقب** التقنية في العراق.
يشرح الموقع خدمة أتمتة إنستغرام الذكية: ردود تلقائية، إدارة طلبات، وسيناريوهات متقدمة.

## الهوية البصرية

- **الألوان:** تدرج أزرق — `#3B82F6` → `#2563EB` → `#1E40AF`
- **الخطوط:** Cairo (العناوين والنصوص العربية) + Inter (العناصر التقنية)
- **الخلفية:** بيضاء وفاتحة — احترافية ونظيفة
